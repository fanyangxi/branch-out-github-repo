# tbd-release-utils:
A util to create `RELEASE` branch on remote github repo.


# Usage:
> tbd-release-utils branch-out --repo "backend-user-api" --branch-name "release"
> tbd-release-utils tag --repo "backend-user-api" --branch-name "release"
